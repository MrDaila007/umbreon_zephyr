# Описание робота Umbreon Zephyr

## Плата управления / контроллер робота

Контроллер робота — **Raspberry Pi Pico 2** на базе микроконтроллера **RP2350A** с двухъядерным процессором ARM Cortex-M33, работающим на частоте 150 МГц. Объём flash-памяти — 4 МБ. Прошивка построена на базе операционной системы реального времени **Zephyr RTOS v4.3**. Используется одно ядро (core 0). Для хранения конфигурационных параметров используется раздел NVS (Non-Volatile Storage) во flash-памяти.

Для связи с внешними устройствами (WiFi-мост) используется модуль **ESP8266**, подключённый по UART1.

## Корпус робота и механика

Робот выполнен в формате автономной гоночной машины (Roborace). Конструкция включает:

- Шасси с рулевым управлением через сервопривод и задним приводом через бесколлекторный мотор с ESC (электронным регулятором скорости).
- Оптический энкодер-тахометр для измерения скорости (диск с 62 отверстиями на оборот).
- OLED-дисплей SSD1306 для отображения параметров и состояния робота.
- Поворотный энкодер с кнопкой для навигации по меню настроек.
- Питание от 2S LiPo аккумулятора (7.4 В номинал) с мониторингом напряжения через делитель (18 кОм / 10 кОм) и отсечкой при 6.0 В.

## Датчики и сенсорные устройства

### Дальномеры

6 датчиков времени пролёта **ST VL53L0X (ToF)** на шине I2C1 (100 кГц). Каждый датчик управляется индивидуальным выводом XSHUT для последовательной инициализации и назначения уникальных I2C-адресов. Рабочий диапазон измерений: 30–2000 мм. Расположение датчиков (вид сверху):

```
        [1:FR]  [4:FL]
  [0:HR]                [5:HL]
        [2:R]   [3:L]
      ────── FRONT ──────
```

- **HR** (Hard-Right) — крайний правый, XSHUT GP6
- **FR** (Front-Right) — передний правый, XSHUT GP7
- **R** (Right) — правый, XSHUT GP8
- **L** (Left) — левый, XSHUT GP9
- **FL** (Front-Left) — передний левый, XSHUT GP14
- **HL** (Hard-Left) — крайний левый, XSHUT GP15

### Гироскоп

**MPU-6050** (IMU) на шине I2C0 (400 кГц), адрес 0x68. Используется только ось Z гироскопа для измерения угловой скорости рыскания (yaw rate). Диапазон: ±500°/с. При запуске выполняется калибровка по 200 замерам (~1 секунда).

### Тахометр

Оптический энкодер на GPIO GP13 (прерывание по переднему фронту). Диск с 62 отверстиями на оборот. Окно подавления дребезга: 200 мкс. Скорость вычисляется по формуле: `v = π × D_колеса / (N_отверстий × T_импульса)`.

### Монитор напряжения батареи

АЦП канал 0 на GP26. Резистивный делитель 18 кОм / 10 кОм (коэффициент 2.8). 12-битный АЦП, опорное напряжение 3.3 В. Порог отсечки: 6.0 В (настраиваемый), срабатывает при удержании ~10 секунд.

## Исполнительные устройства

### Рулевой сервопривод

Рулевой сервопривод управляется ШИМ-сигналом на GP10 (PWM slice 5A):

- Частота: 50 Гц (период 20 мс)
- Диапазон импульсов: 700–2300 мкс
- Управление: значение от -1000 (полностью влево) до +1000 (полностью вправо)
- Механические пределы по умолчанию: 40° мин, 90° нейтраль, 140° макс

### Бесколлекторный мотор с ESC

Бесколлекторный мотор управляется через ESC (электронный регулятор скорости) по стандартному RC-протоколу (PPM/PWM), GP11 (PWM slice 5B):

- Частота: 50 Гц (период 20 мс)
- Нейтраль: 1500 мкс (мотор выключен)
- Вперёд: 1520–1700 мкс (настраиваемый диапазон через параметры MSP/XSP)
- Назад: 1460 мкс (настраиваемый через параметр BSP)
- При первом запуске выполняется калибровка ESC: 2000 → 1000 → 1500 мкс

Отдельный драйвер двигателя не требуется — ESC содержит встроенный драйвер.

## Алгоритм управления роботом

Робот использует **реактивный алгоритм следования вдоль стен** с **PID-регулятором скорости**.

### Управление рулём (следование вдоль стен)

- Величина руления рассчитывается как разность показаний правого и левого датчиков, умноженная на коэффициент: `steer = (sensor[R] - sensor[L]) × coef`.
- Крайние датчики (HR, HL) подмешиваются с весом 25%.
- Используются два адаптивных коэффициента:
  - **COE1** (по умолчанию 0.28) — плавные повороты при свободной трассе (фронт ≥ FOD).
  - **COE2** (по умолчанию 0.65) — резкие повороты при обнаружении препятствия впереди (фронт < FOD).
- При открытом пространстве с обеих сторон — смещение к правой стене (bias 800).

### PID-регулятор скорости

- Частота управляющего цикла: 25 Гц (каждые 40 мс).
- Скорость с энкодера фильтруется экспоненциальным скользящим средним (α = 0.7).
- Ошибка: `error = целевая_скорость - фильтрованная_скорость`.
- Интегральная составляющая ограничена ±50; дифференциальная — по измерению.
- Два режима скорости: SPD1 (~0.48 м/с) — круиз; SPD2 (~0.32 м/с) — замедление в поворотах.

### Детекция застревания

При фронтальном расстоянии < CFD и скорости ≈ 0 счётчик инкрементируется. При превышении порога STK (25 итераций ≈ 1 сек) запускается маневр выхода: стоп → короткий реверс (200 мс) → длинный реверс с максимальным поворотом руля (1800 мс) → продолжение движения.

### Детекция неверного направления

IMU отслеживает накопленный курс. Если робот развернулся более чем на WDD (120°) в направлении, противоположном направлению гонки, запускается аварийное восстановление: стоп → сброс курса → реверс с корректирующим поворотом (1500 мс) → продолжение движения.
